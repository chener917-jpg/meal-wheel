# 今天吃什么

一个静态网页转盘，用来随机决定每天吃什么。

## 发布到 GitHub Pages

1. 在 GitHub 新建一个公开仓库，例如 `meal-wheel`。
2. 上传这些文件到仓库根目录：
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. 打开仓库的 `Settings` -> `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. Branch 选择 `main`，文件夹选择 `/ (root)`，保存。
6. 等待 1-2 分钟，GitHub 会给出固定网址。

## 本地打开

直接双击 `index.html` 即可使用。
